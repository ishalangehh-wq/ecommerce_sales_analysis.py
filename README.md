# 🛒 E-Commerce Sales Analysis

## Project Overview
This project analyzes e-commerce sales data to identify revenue 
drivers, top-performing regions, and high-value customers — 
enabling data-driven business decisions.

**Tools Used:** Python · Pandas · SQL · Matplotlib · Tableau Public

---

## 🔍 Key Insights
- **Laptop** generated the highest revenue (₹1,53,000) among all products
- **North region** contributed the most to overall sales
- **Customer A** was identified as the top buyer by total purchase value
- Sales trend analysis revealed consistent growth patterns over time

## 🛠️ Python Analysis
- Data loading and exploration using **Pandas**
- Product-wise and region-wise sales aggregation using `groupby()`
- Top customer identification using `sort_values()`
- Visualizations using **Matplotlib** — bar charts, pie chart, line chart
